## 🚀 Key Features & Architectural Accomplishments

* **True Decoupled Architecture:** Built a strict "Separation of Concerns" where the frontend on Vercel acts purely as a static interface, passing secure AJAX `fetch` requests across the internet to a totally independent Node.js server hosted on Render.
* **Complex Data Overrides:** Custom server logic on Express intercepts standard text payloads. If it detects the precise Mastermind payload, it bypasses default database values and returns an instant global alert.
* **Database Persistence:** Real-time data logging connecting directly to a cluster in the cloud. Every payload fired from the command panel is securely written to MongoDB.
* **Custom Cyber-Biosecurity Narrative:** A creative storytelling project mapped over a fully functional real-world network infrastructure.
